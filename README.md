# Data Analytics Dashboard

This project is a Sales Data Analytics and Visualization Dashboard using Python.

## Technologies Used
- Python
- Pandas
- Seaborn
- Streamlit

## Features
- Total sales calculation
- Total profit calculation
- Total orders calculation
- Monthly sales bar chart
- Profit trend line chart
- Sales vs orders comparison

## Run Project

```bash
pip install pandas seaborn streamlit
streamlit run app.py
